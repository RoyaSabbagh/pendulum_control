# Summary: #

This package simulates an inverted pendulum connected to a motor with a fixed arm and controls it using torque into a stable upright position.

## Structure: ##

* Nodes:
  - Simulator
  - Controller:

* Massages:
  - control_input.msg
  - Visualization_msgs

* Note:

## Launch: ##

`roslaunch pendulum_control control.launch`
